# 📊 Business Insights 360 – Power BI Dashboard

An end-to-end Power BI project built to analyze business performance across Finance, Sales, Marketing, and Supply Chain domains using real-world business metrics.

---

## 🚀 Project Overview

This project focuses on transforming Excel-based reporting into an interactive BI solution for better decision-making.

The dashboard enables stakeholders to:
- Track financial performance (P&L, Net Sales, GM%)
- Analyze customer and product performance
- Monitor supply chain risks (Forecast Accuracy & Net Error)
- Get executive-level insights in a single view
